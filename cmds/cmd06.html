Diagnostic Commands¶

yo cache-clean¶

Clear Yo’s caches – a good first troubleshooting step.

usage: yo cache-clean [-h]

yo debug¶

Open up a python prompt in the context of a command.

usage: yo debug [-h]

yo help¶

Show help for yo.

usage: yo help [-h]

yo version¶

Show the version of yo and check for updates.

usage: yo version [-h]
